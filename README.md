# 🍔 Ranch Burger

Landing page responsiva desenvolvida para uma hamburgueria temática inspirada no estilo western.

O projeto foi criado com foco em responsividade, organização visual e utilização dos principais componentes do Bootstrap, transformando um exercício acadêmico em um projeto de portfólio.

## 🚀 Demonstração

🔗 Deploy: https://ranch-burger.vercel.app/

## 📸 Preview

![Preview do projeto](./preview-ranch.png)

## ✨ Funcionalidades

* Navbar responsiva
* Menu hamburguer para dispositivos móveis
* Carousel de imagens
* Navegação por seções
* Cardápio organizado por categorias utilizando Tabs do Bootstrap
* Layout responsivo para desktop, tablet e smartphone
* Formulário de contato
* Ícones com Bootstrap Icons
* Identidade visual personalizada inspirada no universo western

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Estruturação semântica com HTML
* Criação de layouts responsivos
* Utilização do Grid System do Bootstrap
* Componentes interativos do Bootstrap
* Organização de arquivos e imagens
* Personalização visual utilizando CSS
* Adaptação de um template base para uma identidade própria

## 📱 Responsividade

O projeto foi desenvolvido seguindo a abordagem Mobile First, garantindo boa experiência em:

* Smartphones
* Tablets
* Notebooks
* Monitores Desktop

## 🎯 Objetivo

Este projeto faz parte da minha jornada de aprendizado em Desenvolvimento Front-end através do curso Desenvolvedor Full Stack Java da EBAC.

O objetivo foi aplicar conceitos de Bootstrap, responsividade e organização visual em um projeto mais próximo de um cenário real de mercado.
